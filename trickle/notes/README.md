# Sistema de Administración

Sistema de gestión con control de roles y usuarios desarrollado con diseño minimalista en blanco y negro.

## Características

- **Autenticación**: Sistema de login con control de acceso
- **Roles de Usuario**: Superusuario, Administrador y Usuario
- **Dashboard**: Panel principal con métricas y gráficos
- **Gestión de Usuarios**: Administración de usuarios del sistema
- **Configuración**: Panel de configuración del sistema
- **Reportes**: Generación y visualización de reportes

## Estructura del Proyecto

- `index.html` - Página de login
- `dashboard.html` - Panel principal del sistema
- `components/` - Componentes React reutilizables
- `utils/` - Funciones auxiliares y autenticación

## Diseño

El sistema utiliza un diseño minimalista con:
- Colores: Negro (#000000) y Blanco (#ffffff)
- Tipografía: Inter, sistema sans-serif
- Componentes: Cards, tablas, gráficos con Chart.js
- Layout: Sidebar lateral + Topbar superior
